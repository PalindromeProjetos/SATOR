## Unidade de Medida para Insumos
 - UnidadeMedida
    - Id:integer;
    - Sigla: string(20);
    - Nome: string(60);
    - UnidadeBase: string(60);
    - MedidaBase: decimal(18,3);
    - Embalagem: string(60);
    - Ativo: boolean;

Exemplo:

    Id: 1;
    Sigla: GL
    Nome: Galão
    UnidadeBase: Litro
    MedidaBase: 5,000
    Embalaagem: Galao
    Ativo: 1
